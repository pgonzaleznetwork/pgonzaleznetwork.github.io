Hi there

Thanks for your support using HappySoup.io.

Over the past few months, Heroku has been forcing more and more updates on the underlying database infrastructure that HappySoup relies on. Many of these updates require significant code and logic changes to the app. 

These days I don't have the bandwidth or sometimes the technical expertise to keep up with the required updates; I'm finding it more and more difficult.

For these reasons, I'm keeping the service offline for the time being.

If you need impact analysis or "where is this used?" functionality

While I figure out if/when/how to bring the app back in a sustainable way, I recommend you use Salto's 100% free tier. I've worked with the team at Salto for the past year to ensure that many of the features you know and love about HappySoup are also available in Salto, free of cost.

The free-forever tier includes:

  • "where is this used?" for any metadata type and any field (custom or standard)
  • Full text search across all metadata, so you can find hardcoded IDs and email addresses
  • Beautiful no-code metadata visualization
